# 工具调用与工作流学习路线

## 1. 学习目标
掌握 Agent 如何安全、稳定地调用外部工具，并串起完整执行流程。

## 2. 核心知识点
- Function Calling / Tool Use 原理
- 工具参数设计与约束
- 状态机与工作流节点
- 异常处理、重试、回滚
- 外部系统交互：文件、接口、数据库、浏览器

## 3. 学习顺序
### 阶段一：工具基础
- 工具描述如何影响模型选择
- 参数 Schema 如何设计
- 工具调用结果如何回传模型

### 阶段二：执行流设计
- 顺序执行
- 条件分支
- 循环与终止条件

### 阶段三：工程稳定性
- 工具失败后的处理策略
- 幂等性与重复执行问题
- 超时与权限控制

## 4. 建议产出
- 封装 3 个示例工具
- 画一个 Agent 工作流图
- 实现一个“查询 + 分析 + 输出”的最小闭环
