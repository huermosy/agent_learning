# 安全与护栏学习路线

## 1. 学习目标
理解 Agent 系统常见风险，并学会通过权限、校验、隔离与审计构建安全边界。

## 2. 核心知识点
- Prompt Injection 与越权诱导
- 工具权限最小化
- 高风险操作确认机制
- 输入输出过滤
- 审计日志与责任追踪

## 3. 学习顺序
### 阶段一：风险识别
- LLM 特有安全问题
- Agent 与传统 Web 安全的不同点
- 典型攻击面整理

### 阶段二：护栏设计
- 系统提示词护栏
- 工具调用白名单
- 人工确认节点

### 阶段三：治理与审计
- 日志留存
- 失败复盘
- 安全策略迭代

## 4. 建议产出
- 做一份 Agent 风险清单
- 设计一个敏感操作审批流
- 给现有 Agent 方案补一版安全设计
