# RAG 学习路线

## 1. 学习目标
掌握 RAG 的完整链路，理解知识是如何进入系统、如何被召回、如何参与生成，以及如何与 Agent 的规划和工具调用能力结合。

## 2. 核心知识点
- RAG 基本架构：索引、检索、增强生成
- 文档清洗、切分与 chunk 策略
- Embedding、向量数据库、召回策略
- 重排（Rerank）与上下文构造
- RAG 的评估与幻觉控制
- Agent + RAG 的边界与组合方式

## 3. 学习顺序
### 阶段一：基础认知
- 为什么需要 RAG
- RAG 和微调的区别
- 基础检索流程

### 阶段二：工程实现
- 文档切分策略
- 检索召回与 Top-K
- 重排与上下文拼装

### 阶段三：系统优化
- 召回率与准确率平衡
- 降低幻觉与无关上下文
- 与 Agent 规划、记忆、工具调用集成

## 4. 建议产出
- 搭建一个最小 RAG Demo
- 对比不同 chunk 策略的效果
- 设计一个 Agent + RAG 组合案例

## 5. 进阶衔接
完成后可回到：`../05-Multi-Agent-Systems/README.md`，继续学习复杂 Agent 系统如何消费检索能力。
