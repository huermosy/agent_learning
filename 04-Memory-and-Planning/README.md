# 记忆与规划

## 模块简介
本模块关注 Agent 如何记住信息、如何做计划、如何把复杂任务拆成可执行步骤，并在执行中动态修正。

## 你将学到
- 短期/长期记忆设计
- 任务分解与计划生成
- ReAct / Plan-and-Execute 等模式
- 反思、自检与迭代执行
