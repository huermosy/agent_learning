# 记忆与规划学习路线

## 1. 学习目标
理解 Agent 如何记住上下文、拆解任务、生成计划并在执行中修正自己。

## 2. 核心知识点
- 短期记忆：会话上下文、状态缓存
- 长期记忆：用户偏好、知识沉淀、向量记忆
- 任务规划：分解、排序、依赖关系
- 执行反馈：反思、纠错、自检

## 3. 学习顺序
### 阶段一：记忆模型
- 什么信息该记
- 什么信息不该记
- Memory 的生命周期设计

### 阶段二：规划模型
- 直接执行 vs 先规划后执行
- 大任务拆分方法
- 计划更新机制

### 阶段三：自我修正
- Reflection
- Critic / Verifier
- 自检闭环

## 4. 建议产出
- 设计一份 Agent memory schema
- 写一个任务分解模板
- 实现一个“计划-执行-检查”示例

## 5. 进阶衔接
结合已有 `RAG` 目录，思考记忆与检索的边界，再进入多 Agent 模块。
