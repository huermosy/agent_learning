# Agent 基础

## 模块简介
本模块帮助你建立 Agent 开发的整体心智模型，理解 Agent 与传统应用、工作流自动化、RAG 系统之间的关系。

## 你将学到
- Agent 的定义与边界
- 单轮调用、多轮对话、工具增强 Agent 的区别
- Agent 运行时的基本结构
- 常见 Agent 模式与落地场景

## 适合当前模块的人
- 刚进入 Agent 开发领域
- 已做过 Prompt 工程但缺乏系统框架
- 想把 RAG 扩展为 Agent 系统
