# Agent 基础学习路线

## 1. 学习目标
建立 Agent 开发的基础认知，理解它和脚本、工作流、RAG、聊天机器人的差异。

## 2. 核心知识点
- Agent 的定义：感知、决策、行动、反馈
- Agent 与 Workflow 的区别
- 单 Agent / 多 Agent 基本形态
- Agent 系统的输入、状态、输出
- Agent 应用常见场景：问答、执行、分析、编排

## 3. 学习顺序
### 阶段一：概念入门
- 什么是 Agent
- 为什么 LLM 让 Agent 成为主流
- Agent 的组成部分

### 阶段二：模式识别
- ReAct
- Plan-and-Execute
- Tool-Using Agent
- Reflection / Self-Critique

### 阶段三：案例理解
- 简单问答 Agent
- 带工具调用的任务 Agent
- 带检索增强的知识型 Agent
