# Agent 基础术语表

## 核心术语
- **Agent**：能够基于目标感知、推理、行动并反馈修正的系统。
- **Workflow**：预定义流程驱动的任务执行方式。
- **ReAct**：推理与行动交替进行的 Agent 模式。
- **Tool Use**：模型通过工具与外部环境交互的能力。
- **Reflection**：模型对自身结果进行复查和修正的机制。

## 常见易混淆概念
- **Agent vs Chatbot**：Chatbot 偏对话，Agent 偏任务完成。
- **Agent vs Workflow**：Agent 更灵活，Workflow 更稳定可控。
