# 多 Agent 系统学习路线

## 1. 学习目标
掌握多个 Agent 之间如何分工、协作、竞争与监督，构建复杂任务系统。

## 2. 核心知识点
- Router / Supervisor / Specialist 模式
- 共享状态与消息传递
- 任务分配与结果汇总
- 并行执行与冲突控制
- 成本、时延与收益权衡

## 3. 学习顺序
### 阶段一：模式理解
- 单 Agent 的瓶颈
- 为什么需要多 Agent
- 常见多 Agent 组织方式

### 阶段二：协作设计
- 子任务切分
- Agent 间通信协议
- 仲裁与审批节点

### 阶段三：系统优化
- 减少重复劳动
- 控制上下文膨胀
- 降低协作噪音

## 4. 建议产出
- 设计一个 3 角色多 Agent 系统
- 给每个角色写清楚职责边界
- 做一次单 Agent 与多 Agent 的效果对比
