# 多 Agent 系统术语表

## 核心术语
- **Supervisor**：负责任务调度与监督的上层 Agent。
- **Router**：负责把任务路由给合适子 Agent 的组件。
- **Specialist**：面向特定能力域的专门 Agent。
- **Shared Context**：多个 Agent 共享的上下文或状态。
- **Arbitration**：多个候选结果之间的仲裁机制。

## 常见易混淆概念
- **Router vs Supervisor**：前者偏分发，后者偏统筹与监督。
- **多 Agent vs 多工具**：前者是多角色协作，后者是单 Agent 调多工具。
