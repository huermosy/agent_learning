# 多 Agent 系统

## 模块简介
本模块研究多个 Agent 如何分工协作，包括角色设计、任务路由、监督者模式与并行执行策略。

## 你将学到
- 多 Agent 架构模式
- Supervisor / Router / Specialist 设计
- 通信协议与共享上下文
- 并发、冲突与协作控制
