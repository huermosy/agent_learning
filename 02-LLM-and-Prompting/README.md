# LLM 与 Prompt 工程

## 模块简介
本模块关注 Agent 的“大脑层”，学习如何通过 Prompt、上下文组织、结构化输出与推理控制提升 Agent 的稳定性与可控性。

## 你将学到
- Prompt 设计原则
- 上下文窗口管理
- 输出格式约束
- 思维链、计划先行与推理控制
