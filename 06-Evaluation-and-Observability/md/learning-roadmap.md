# 评估与可观测性学习路线

## 1. 学习目标
掌握如何系统评估 Agent 质量，并通过日志、轨迹、指标持续优化系统。

## 2. 核心知识点
- 离线评测与在线评测
- 任务成功率、成本、延迟
- 日志、Trace、Span、轨迹回放
- 人工评审与自动评分
- Prompt / Tool / Workflow 的 A/B 对比

## 3. 学习顺序
### 阶段一：评估基础
- 什么叫“Agent 做对了”
- 结果评估 vs 过程评估
- 主观质量如何量化

### 阶段二：观测体系
- 日志字段设计
- Trace 链路记录
- 关键事件埋点

### 阶段三：优化闭环
- 从失败案例反推问题来源
- 构造 benchmark
- 用数据驱动迭代

## 4. 建议产出
- 定义一份 Agent 评估指标表
- 为一个 Agent 项目设计 Trace 结构
- 整理 10 条失败样本并归因
