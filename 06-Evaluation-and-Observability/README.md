# 评估与可观测性

## 模块简介
本模块关注如何验证 Agent 是否真的有效，包括评测体系、日志、轨迹追踪、人工反馈与线上质量监控。

## 你将学到
- Agent 评测维度
- 轨迹与日志设计
- 成功率、成本、时延指标
- 人工评估与自动评估结合
