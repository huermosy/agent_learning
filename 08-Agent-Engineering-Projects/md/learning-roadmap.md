# Agent 工程实战学习路线

## 1. 学习目标
把基础、模型、工具、记忆、评估、安全组合起来，形成完整的 Agent 工程落地能力。

## 2. 项目能力清单
- 需求分析与范围控制
- 架构设计与模块拆分
- 工具与数据源接入
- 评估与监控埋点
- 安全边界与上线策略

## 3. 推荐实战路径
### 项目一：知识问答 Agent
- 结合已有 RAG 能力
- 增加工具调用与结果整理
- 补充评估与反馈

### 项目二：任务执行 Agent
- 支持多步骤计划
- 接入外部 API / 文件系统
- 增加失败恢复机制

### 项目三：多 Agent 协作系统
- Router + Specialist + Verifier
- 增加观测与质量评估
- 做工程化目录拆分

## 4. 建议产出
- 完成 1 个最小可运行项目
- 完成 1 个具备评估与日志的中型项目
- 整理 1 套自己的 Agent 工程模板

## 5. 后续网页化方向
- 为每一关增加目标、前置技能、任务、验收标准
- 把 Markdown 转成可交互章节
- 增加 checkpoint 与徽章系统
